# CPP_design

This directory contains two independent projects that can be run separately.

## Projects

### optimization_cycle

A minimal, publication-ready pipeline for iterative optimization (prediction, BO, DiffDock, MD).

- Location: `optimization_cycle/`
- Docs: `optimization_cycle/README.md`

### sequence_generation

Sequence generation and finetuning utilities (EvoDiff) and a minimal CPP classifier.

- Location: `sequence_generation/`
- Docs: `sequence_generation/README.md`
